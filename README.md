# homebrew-murmur

Official Homebrew tap for [Murmur](https://github.com/roshanshah11/murmur), a local-first macOS dictation app.

## Install

```
brew install --cask roshanshah11/murmur/murmur
```

Bugs and feature requests: [main repo](https://github.com/roshanshah11/murmur).

This tap is automatically bumped by the main repo's `release.yml` on every tagged release.
